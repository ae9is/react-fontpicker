# react-fontpicker

A Google font picker component for React.

- No dependencies (other than React)
- 1600+ Google fonts
- Font previews from pre-generated SVG images
- Optionally autoloads fonts

Forked from https://github.com/Mikk3lRo/vue-fontpicker/
